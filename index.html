Question Grouping and Random Selection

üìö What is Grouping?
Grouping organizes related questions together (like "Chapter 1" or "Easy Questions"). This helps structure your quiz and enables random question selection.
‚úÖ How to Use Groups:
  • To group questions: Enter the same group name in column B for all questions you want together
  • To leave ungrouped: Simply leave column B blank - the question stands alone
  • Group names: Use any name like "Chapter 1", "Atoms", "Lab Safety", etc.
üé≤ Random Selection (Optional):
Want Canvas to randomly pick only some questions from a group? Use columns C and D on the first row of each new group:
  • Column C (GroupPick): How many questions to randomly select
  • Column D (GroupPointsPerQuestion): Points each selected question is worth
  • Important: Only fill these in on the first question of a new group, leave blank for other rows

Question type instructions

MC

Single correct choice. Put choices in Answer1..Answer5. Put the correct index in Correct, for example 3. Uses bracket format: [ ] for incorrect, [*] for correct.

MR

Multiple correct choices. Put choices in Answer1..Answer5. Put a comma list in Correct, for example 1,3. Uses bracket format: [ ] for incorrect, [*] for correct.

TF

Use Answer1=True and Answer2=False or your own labels. Put Correct=1 for True or 2 for False.

NUM

Put the rule in Answer1. Examples: 5 for exact; [1.2598, 1.2600] for range; 1.4142 +- 0.0001 or 100 +- 5% for margin.

SA

Put acceptable exact answers in Answer1..Answer5. Case sensitivity follows Canvas behavior.

ESSAY

No answers needed. The generator adds the essay marker automatically.

FILE

No answers needed. The generator adds the file-upload marker automatically.

TEXT

Non-question content. Use TextTitle and TextContent. No points.
